# Universal Framework — the 20 modules

Pipeline; each module = {inputs · owner questions · data · AI output · handoff}. "Same soup, different ingredients" — logic fixed, inputs swap per client.

| # | Module | Owner question | Output → next |
|---|---|---|---|
| 1 | Business Understanding | What/whom/price/margin/cycle/CLV? | model + max CPA → 5,13 |
| 2 | Audience Intelligence | Best past customer? | personas → 3,4,11,13 |
| 3 | Customer Pain Mapping | Pre-purchase complaints? | ranked pains → 7,8 |
| 4 | Customer Desire Mapping | Status/feeling wanted? | desires → 7,8 |
| 5 | Offer Positioning | Dream outcome + belief gap? | offer statement → 7,11,12 |
| 6 | Competitor Intelligence | Who else chases this buyer? | gap map (real Ad-Library teardowns) → 7 |
| 7 | Viral Angle | — | pains×desires×angles → 8 |
| 8 | Hook Generation | — | hooks → 9 |
| 9 | Script Generation | — | scripts → 10 |
| 10 | Creative Direction | Assets on hand? | shot list → produce |
| 11 | Lead Magnet | Smallest buyer-only thing? | magnet spec → 14,16 |
| 12 | Giveaway Campaign | Budget/prize? | prize score + qualifying Qs → 13,14 |
| 13 | Ad Targeting | Budget/country/exclusions? | targeting → 14 |
| 14 | Lead Form | 3 facts that qualify? + consent | form spec → 15,16 |
| 15 | Lead Scoring | — | tiered leads → 16 |
| 16 | Follow-Up | Who responds, how fast? | sequences (HOT <5 min) → 17,18 |
| 17 | Retargeting | — | warm ladders → 18 |
| 18 | Sales Conversion | Objections? | scripts + FNA gate → 19 |
| 19 | Analytics | — | CPL/CPQL/close dashboard → 20 |
| 20 | Iteration | — | re-weight scores → 2,7 |

**Reuse-first:** modules 7–10 run the proven house creative engine (Bitton script formula + retention rules; hook craft + Lego Brick OS), not a generic rebuild. Module 19 reuses an existing first-party funnel + pixel pattern + the CAPI loop (see `04-finance-compliance.md`).

**Intake "ingredients":** company, industry, offer, price, margin, sales cycle, CLV, audience, platform(s), country, conversion goal, offer type, prize/magnet, past customer data, past ad data, competitors, compliance restrictions, brand voice/assets. (Full questionnaire in `05-intake-and-prompts.md`.)
